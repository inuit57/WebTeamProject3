
## 중고거래 사이트
- 국비지원 팀프로젝트 

## 주요 기능 
- 로그인 / 카카오 로그인
- 이메일/휴대전화로 회원 가입 ( 이메일 인증 / 문자 인증 ) 
- 상품 게시판 
  - 최대 4개까지 이미지 등록 및 수정/삭제. 
  - 카테고리/작성자 검색 수행
  - 페이징 기능
  - 채팅 기능(웹소켓) : 채팅 창에서 거래 수행
- 일반 게시판 
  - 게시판 CRUD
  - 댓글 기능 
- 신고 기능 ( 일반/상품 게시판 ) 
- FAQ 게시판 
- 경매 게시판 
- 1:1 문의 게시판
  - 답글 기능
- 관리자 페이지
  - 회원 등급 관리 및 회원 탈퇴 처리 
  - 신고된 글 관련 처리
  - 1:1 문의 처리
- 충전 페이지 
  - 카카오페이 API 
  
## 사용된 API 및 기술 
- 카카오 로그인 API 
- 카카오 페이 
- 휴대폰 문자 인증 API 
- Web Socket
